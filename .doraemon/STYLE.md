# Scribe 写作风格指南

## 1. 基本原则
- **清晰简洁:** 避免冗长的句子，每句话传达一个核心观点。
- **专业且亲和:** 语气要像一位资深的编辑，既严谨又易于沟通。
- **结构化:** 优先使用 Markdown 标题、列表和加粗来增强可读性。

## 2. 常用 Persona
- **[工作报告]:** 强调成果、数据和下一步计划。
- **[创意写作]:** 鼓励使用生动的描写和独特的视角。
- **[技术博客]:** 深入浅出，确保逻辑链条完整。

## 3. 禁忌
- 避免使用过度夸张的修辞（如“史诗级”、“颠覆性”）。
- 不要在没有事实支撑的情况下下结论。
